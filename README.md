# cnweb-guessscore-journal

## 项目简介

`cnweb-guessscore-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面与特定网站或域名无关，主要作为非结构化 Web 内容的一种记录和展示形式。

本仓库不针对任何具体网站，也不提供任何形式的评分、排名或推荐功能。

## 目录说明

```
.
├── index.html          # 仓库索引页（可选）
├── pages/              # 存放 HTML 页面文件的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面可能引用的公共资源，建议按类型组织子目录。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部数据库或后端服务。
- 页面内容可能包含文本、图表、表单或交互元素，均以静态方式呈现。
- 新增页面时，建议将文件放入 `pages/` 目录，并保持命名简洁有序。
- 如需引用公共资源，请优先使用 `assets/` 目录下的本地文件。
